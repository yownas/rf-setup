# rf-setup.ps1

* Download [rf-setup.ps1](https://raw.githubusercontent.com/yownas/rf-setup/refs/heads/main/rf-setup.ps1) and save it in a new folder where you want to install RuinedFooocus.
* Run the script.
* Select python version.
* Install RuinedFoocus by selecting branch (you most likely want the "main" branch)
* Optional: Select Torch version. RuinedFooocus will try to automatically find a good version that fits your GPU, but AMD users might need to force a specific version.
* Optional: Create a run.bat script to start RuinedFooocus
